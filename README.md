<h3>🥩 E-commerce para Açougue</h3>
<p>
  Desenvolvimento de um site interativo para um açougue, focado na experiência 
  do usuário e gestão de vendas. O cliente pode explorar os produtos por 
  departamentos e adicioná-los ao carrinho conforme a disponibilidade em estoque. 
  O sistema conta com um carrinho dinâmico que calcula em tempo real a quantidade 
  total de itens e o valor final. A finalização da compra exige autenticação, 
  redirecionando usuários não logados para login ou cadastro.
</p>
